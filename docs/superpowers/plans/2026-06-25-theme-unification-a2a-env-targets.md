# Theme Unification A2a — env/sync CLI targets Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extend the wana theme pipeline to the env/sync-driven CLI tools — pywal colorschemes, fzf, bat (base16 tmTheme), and delta — including the new shell consumer that applies the active variant to `BAT_THEME` and `FZF_DEFAULT_OPTS`.

**Architecture:** Reuses the A1 pipeline (wana `gen.py` renderers → `theme-sync` vendoring → manifest). These targets don't use the `current`-symlink switch path; they're either sync-only (pywal) or env-variant (bat/fzf), selected at shell startup from `~/.cache/wana/variant`. delta inherits bat's theme via `BAT_THEME`.

**Tech Stack:** Python 3 (wana `gen.py`, `str.format` templates, `unittest`), POSIX sh + fish (shell env consumer), GNU stow deploy.

**Scope:** A2a of three (A2b = herdr/starship/btop/lazygit; A2c = atuin/opencode/yazi + Noctalia cleanup + docs). Spec: `docs/superpowers/specs/2026-06-25-theme-unification-a2-cli-targets-design.md`. Builds on A1 (already merged to main).

**Branch:** before starting, `cd ~/Repos/wana && git checkout -b feat/theme-renderers-a2a` and `cd ~/Repos/dotfiles && git checkout -b feat/theme-a2a-env-targets`.

## Manifest column reference (from A1)

```
# name | wana_src{v} | repo_dest{v} | home_current | home_variant{v} | reload
```

`home_current = -` marks a **sync-only** row: `theme-sync` vendors it (cols 1-3), `theme-switch` skips it (no symlink/reload). pywal and fzf use `-`. bat is vendored as two real files selected by `BAT_THEME`, also `home_current = -`.

```python
def render_bat(s: Scheme) -> tuple[str, str]:
    with open(os.path.join(TPL, "bat.tmTheme.tmpl")) as f:
        tpl = f.read()
    fields = {"name": f"wana-{s.variant}"}
    fields.update({f"base{n:02X}": s.hex(f"base{n:02X}") for n in range(16)})
    out = tpl.format(**fields)
    rel = f"themes/bat/wana-{s.variant}.tmTheme"
    return rel, out


RENDERERS = [render_kitty, render_alacritty, render_tty, render_pywal, render_fzf, render_bat]
```
