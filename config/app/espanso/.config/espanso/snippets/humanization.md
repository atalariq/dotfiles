# Academic Essay Humanization Prompt

You are helping me revise an academic essay.

Your goal is not to bypass AI detectors. Your goal is to make the essay clearer, more natural, more academically defensible, and more visibly owned by the student writer.

Do not merely paraphrase sentence by sentence. Rebuild weak parts from the underlying ideas.

## Context

This essay is for an academic writing class. The final result should sound like a real student essay: clear, thoughtful, moderately formal, and defensible, but not over-polished or artificially sophisticated.

## Main Tasks

1. Diagnose why the current draft feels AI-like, generic, weak, or overly polished.
2. Review whether the essay answers the assignment properly.
3. Check whether the claims match the provided references.
4. Identify paragraphs that need structural rewriting, not just wording changes.
5. Rewrite the essay in a natural academic student voice.
6. Add writer judgment where appropriate.
7. Improve source engagement.
8. Reduce formulaic transitions and generic academic filler.
9. Preserve meaning, citations, and academic integrity.
10. Provide change notes and remaining risks.

## Revision Principles

Do not optimize for AI detector scores.

Instead, improve:

- thesis clarity
- paragraph purpose
- argument flow
- source engagement
- student ownership
- sentence rhythm
- concrete explanation
- defensibility

Avoid rewriting everything into perfect, generic academic prose.

A strong revised essay should sound like the student understands the topic and can explain the argument orally.

## What to Avoid

Avoid:

- sentence-by-sentence paraphrasing
- fake citations
- invented page numbers
- unsupported claims
- overly advanced vocabulary
- robotic transitions
- generic phrases
- over-polished academic tone
- making the essay sound unlike a student

Avoid phrases like:

- "In today's modern world"
- "plays a crucial role"
- "has a significant impact"
- "it is important to note that"
- "various aspects"
- "moreover"
- "furthermore"
- "in conclusion" when used mechanically

Use simpler and more direct transitions when possible.

## Source Engagement Rules

Do not use sources as decoration.

Weak source use:

"According to Smith, technology improves learning."

Better source use:

"Smith's argument is useful because it shows that technology does not simply make learning faster; it also changes where and when learning can happen."

When appropriate, engage with sources by:

- agreeing
- extending
- contrasting
- limiting
- applying
- questioning

Flag any claim that is not supported by the provided references.

Never invent sources, quotes, findings, statistics, or page numbers.

## Writer Ownership

Where suitable, add controlled student judgment.

Examples:

- "For students like me, this means..."
- "This point matters because..."
- "This argument is useful, but it does not fully explain..."
- "In practice, the issue is not only..., but also..."
- "This suggests that..."

Do not overuse "I". Use it only when it fits the assignment and essay style.

## Paragraph Rewrite Method

For weak paragraphs, rebuild using this structure:

- Main claim:
- Evidence:
- Student interpretation:
- Why it matters:
- Limitation or nuance:
- Link to next idea:

Then rewrite the paragraph naturally.

## Required Output

Produce the response in this format:

### A. Diagnosis

Explain briefly what feels weak, generic, AI-like, or over-polished.

### B. Content Review

Review:

- thesis
- argument flow
- paragraph coherence
- evidence use
- source alignment
- assignment fit

### C. Revision Plan

List the main changes needed.

### D. Revised Essay

Rewrite the essay or the requested section.

### E. Change Notes

Explain what changed and why.

### F. Remaining Risks

Mention issues such as:

- missing references
- weak evidence
- uncertain citation support
- word count mismatch
- rubric mismatch
- parts that still need the student's own input
